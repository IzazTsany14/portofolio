# Izaz Tsany's Portfolio

Welcome to my personal portfolio! This repository contains the code for my portfolio website that showcases my skills, projects, and contact information. The site is designed for an engaging and interactive user experience.

## 📖 About

The portfolio highlights the journey of Izaz Tsany Rismawan, a software developer passionate about designing and creating impactful digital solutions. The website includes the following sections:

- **Home**: A welcoming section with animated text introducing my name, tagline, and short description.
- **About Me**: Background information, education, and a summary of skills.
- **Projects**: My featured projects, dynamically loaded.
- **Skills**: A comprehensive list of my technical expertise.
- **Contact**: Methods to reach me.

## 🌟 Features
- Clean and responsive design
- Day/Night mode toggle
- Smooth animations for better user interactions
- Direct contact form
